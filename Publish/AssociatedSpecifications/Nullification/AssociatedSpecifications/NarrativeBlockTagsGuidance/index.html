About

Overview

This pack gives guidance on the use of tags and attributes used in a CDA text section (also called CDA markup).

The content of this pack is based on the schema "NarrativeBlock.xsd", which is included in Domain Message Specifications in the "dt" folder. This schema is included in the Generic CDA and Profile schemas to ensure the correct CDA mark-up has been applied to the <text> element within the section class.

This pack contains:

  • A description of the tags allowed in a text section
  • Allowable nesting of tags
  • Description of attributes that may be used with each tag
  • An example for each tag
  • Further examples for common styles used in CDA documents
