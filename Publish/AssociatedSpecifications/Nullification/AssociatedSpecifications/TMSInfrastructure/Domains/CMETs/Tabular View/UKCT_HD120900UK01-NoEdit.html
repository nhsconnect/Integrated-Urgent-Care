AgentPersonSDS

Identifies a person fulfilling a specific role where the Person, Role Profile and Job Role are all mandatory. Full details of the person, role profile and job role are available from SDS.
Used by:
[1..1] classCode (CS { CNE:AGNT} ){ } Fixed="AGNT"

[1..1] id (II)

An SDS identifier that uniquely identifies the Role of the Person. This is carried in the following way:

  • The root attribute shall contain an OID with the value 1.2.826.0.1285.0.2.0.67
  • the extension attribute shall contain the SDS Role Profile ID, known in the External Interface Specification as Role Profile Code
