HL7StandardVersionCode

Code SystemId:
Version: 32.0
Status: active
Date: 2013-07-11

Description

Describes the version of HL7 V3 used to format the messages in a specific version of the message specification.

Note: as of 11/07/2013 this vocabulary will no longer be updated in the infrastructure message specification. The code to be used for the given domain should be taken from the version of this vocabulary contained in the domain message specification. The code will be enumerated in the InfVocab.xsd contained in the vocabulary schema folder within the domain message specification.
