CDADocumentTypeSnCT

Code SystemId: 2.16.840.1.113883.2.1.3.2.4.15
Version: 2.0
Status: active
Date: 2007-04-16

Description

A subset of the Snomed-CT clinical terminology (i.e. coding system) intended for use in the UK (previously named SnomedCTUK) that describes a type of CDA document.

Any code from the SNOMED CT UK CDA Document Type subset with subset original id 1281000000131.
