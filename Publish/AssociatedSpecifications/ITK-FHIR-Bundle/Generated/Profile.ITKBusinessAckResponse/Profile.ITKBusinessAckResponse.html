    The following structure definitions are available in this profile.

    NameTypeDescription & Constraints
    ITK-Message-Bundle-1BundleITK message bundle where the first resource is a message header.
    ITK-MessageHeader-1MessageHeaderProfile to define the message header use within an ITK bundle.
    CareConnect-Practitioner-1PractitionerThe Practitioner resource represents the healthcare professional directly or indirectly involved in the provision of healthcare related services.
    ITK-BusinessAck-OperationOutcome-1OperationOutcomeProfile to define an business acknowledgment for an ITK FHIR message.
    CareConnect-Organization-1OrganizationThe Organization resource represents the organisation that employs the healthcare professional.
    CareConnect-Location-1LocationThe Location resource provides information and details on the physical location and the services provided.
