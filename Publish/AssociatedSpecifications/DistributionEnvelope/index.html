Overview: Distribution Envelope Specification

The Distribution Envelope provides a standardised structure to carry information regarding common end-to-end distribution requirements.  The Distribution Envelope is used to wrap one or more payloads in an ITK interaction.  It is independent of any lower-level transport protocol, and provides a set of features including:

  • End-to-end message identification
  • Declaration of original service name, preserved end-to-end
  • Addressing information about the recipient(s)
  • Information about the sender, for auditing
  • Information (“metadata”) about the payload(s)

For further details on the use of the Distribution Envelope, consult the ITK Core Pack (available on TRUD).
