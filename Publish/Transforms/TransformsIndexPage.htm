Transforms


Transform Description
TrueCDAToCDALike_v2.xsl When creating a message to the "on-the-wire" CDA format, this transform will take the "on-the-wire" format XML instance and create a "templated" version of the same instance. The "templated" version can then be schema validated against the NHS 111 CDA conformance schema provided in the Schemas folder within this pack (POCD_MT200001GB02.xsd). If the XML instance validates against the CDA conformance schema, a valid NHS 111 instance has been created.
CDA_NPfIT_Document_Renderer.xsl This is a sample transform that renders a valid CDA XML instance into a html document. This transform is an example transform only and is not expected to be used "as is" in  production systems.
